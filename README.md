# No Look Pass Robot
터틀봇과 키넥트 그리고 ROS Framework 이용하여 자동으로 장애물을 피하고 주인을 추종하기 위해 시작한 프로젝트.\
장애물 회피는 가능하지만, 주인 추종에 쓰기 위해 딥러닝을 사용해보려 했으나 너무 많은 리소스가 필요해\
장애물을 자동으로 계속 피하면서 주행하도록 했습니다.
